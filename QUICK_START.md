# Eigent Chrome Extension - Quick Start Guide

## Installation (5 minutes)

### Step 1: Open Extension Management
- **Chrome**: Navigate to `chrome://extensions`
- **Brave**: Navigate to `brave://extensions`

### Step 2: Enable Developer Mode
- Toggle the switch in the top-right corner
- It should say "Developer mode is enabled"

### Step 3: Load Extension
1. Click "Load unpacked" button
2. Navigate to: `/Users/Subho/eigent-chrome-extension`
3. Select the folder and click "Select"

### Step 4: Configure API Key
1. Click the Eigent icon in your browser toolbar (puzzle piece → Eigent)
2. Click the ⚙️ settings icon
3. Add your **Anthropic API Key** (get one at https://console.anthropic.com)
4. Click "Test Connection" 
5. Click "Save Settings"

## First Task

Try this example task:

1. Click the Eigent extension icon
2. Make sure "Multi-Modal" agent is selected
3. Enter task: "Summarize the benefits of multi-agent AI systems"
4. Click "Execute Task"
5. Watch the agents work!

## Troubleshooting

**Extension not showing?**
- Make sure Developer Mode is ON
- Try removing and re-adding the extension
- Check for errors in browser console

**API errors?**
- Verify your API key is correct
- Check you have credits at console.anthropic.com
- Try the "Test Connection" button in settings

**Nothing happens?**
- Open browser DevTools (F12) and check Console tab
- Look for error messages
- Ensure all agents are enabled in settings
